## Features

-  Auto-detects user location using Geolocation API  
-  Search weather by city name  
-  Displays:
  - Temperature
  - Feels like
  - Humidity
  - Wind speed
  - Pressure
  - Visibility
  - Min / Max temperature  
- Sunrise & Sunset time  
- Line chart for next hours temperature (Chart.js)  
- Hourly forecast cards with icons  
- Clean and responsive UI
